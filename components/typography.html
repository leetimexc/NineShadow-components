Typography 排版 ​

用户展示段落、标题、文本内容

标题 ​

通过 level 展示不同级别的标题,一共六个级别

<template>
  <ns-title :level="1">H1 标题</ns-title>
  <ns-title :level="2">H2 弹窗标题</ns-title>
  <ns-title :level="3">H3 提示信息标题</ns-title>
  <ns-title :level="4">H4 菜单标题</ns-title>
  <ns-title :level="5">H5 段落标题</ns-title>
  <ns-title :level="6">H6 导航分类</ns-title>
</template>

类型 ​

所有文本内容都有该属性，这里以标题为例。不同类型分别有：primary,info,success,warning,danger 五种类型。

<template>
  <ns-title :level="3">标题不同类型展示</ns-title>
  <ns-title :level="3" type="info">标题不同类型展示</ns-title>
  <ns-title :level="3" type="primary">标题不同类型展示</ns-title>
  <ns-title :level="3" type="success">标题不同类型展示</ns-title>
  <ns-title :level="3" type="warning">标题不同类型展示</ns-title>
  <ns-title :level="3" type="danger">标题不同类型展示</ns-title>
</template>

段落省略 ​

使用 rows 限制段落显示的行数。

<template>
  <ns-switch v-model="isOpen"></ns-switch>
  <ns-paragraph
    v-model:open="isOpen"
    moreable
    :rows="2"
    @change-open="handleOpen"
  >
    Terraria is the land of adventure! It’s a mysterious place! It is a land for
    you to shape, defend and enjoy. In Terraria, your options are endless. Fan
    of action games with itchy fingers?
  </ns-paragraph>

  <p>{{ status }}</p>
</template>

<script setup="ts">
import { ref } from 'vue'

const isOpen = ref(false)
const status = ref(false)

function handleOpen(value) {
  status.value = `open 状态：${value}`
}
</script>
