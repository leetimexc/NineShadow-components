NsSpace 间距 ​

基础用法 ​

<template>
  <ns-space>
    <ns-button>按钮1</ns-button>
    <ns-button>按钮2</ns-button>
    <ns-button shape="circle"><IconPlusOutline /></ns-button>
  </ns-space>
</template>

间距 ​

间距 您可以通过 size 来设置间距，可以是数字或内置尺寸（ss、s、m、l、xl），也可以是数组，分别对应水平和垂直间距。

<template>
  <p>内置尺寸：</p>
  <ns-radio-group v-model="size" type="button">
    <ns-radio value="ss">ss</ns-radio>
    <ns-radio value="s">s</ns-radio>
    <ns-radio value="m">m</ns-radio>
    <ns-radio value="l">l</ns-radio>
    <ns-radio value="xl">xl</ns-radio>
  </ns-radio-group>
  <ns-space :size="size" class="margin">
    <ns-button>按钮1</ns-button>
    <ns-button>按钮2</ns-button>
    <ns-button>按钮3</ns-button>
  </ns-space>

  <p>
    自定义间距：
    <ns-input v-model="inputSize" style="width: 80px" />
  </p>
  <ns-space :size="`${inputSize}px`" class="margin">
    <ns-button>按钮1</ns-button>
    <ns-button>按钮2</ns-button>
    <ns-button>按钮3</ns-button>
  </ns-space>

  <p>自定义水平和竖直间距：[10, 30]</p>
  <ns-space wrap :size="[10, 20]" class="margin" style="width: 300px">
    <ns-button v-for="item in 10" :key="item">按钮</ns-button>
  </ns-space>
</template>

<script lang="ts" setup>
import { ref } from 'vue'

const size = ref('l') as any
const inputSize = ref('10') as any
</script>

<style>
.margin {
  margin: 12px 0 24px;
}
</style>

水平对齐 ​

您可以通过 justify 属性来设置 y 轴对齐方式。可选值：start、center、end、between、around。默认为 start。

<template>
  <ns-radio-group v-model="justify" type="button">
    <ns-radio value="start">start</ns-radio>
    <ns-radio value="center">center</ns-radio>
    <ns-radio value="end">end</ns-radio>
    <ns-radio value="between">between</ns-radio>
    <ns-radio value="around">around</ns-radio>
    <ns-radio value="evenly">evenly</ns-radio>
  </ns-radio-group>
  <ns-space :justify="justify" style="margin-top: 12px">
    <ns-button>按钮1</ns-button>
    <ns-button>按钮2</ns-button>
    <ns-button shape="circle"><IconPlusOutline /></ns-button>
  </ns-space>
</template>

<script lang="ts" setup>
import { ref } from 'vue'

const justify = ref('start') as any
</script>

垂直对齐 ​

您可以通过 align 属性来设置 x 轴对齐方式。可选值：start、center、end、baseline。默认为 start。

<template>
  <ns-radio-group v-model="align" type="button">
    <ns-radio value="start">start</ns-radio>
    <ns-radio value="center">center</ns-radio>
    <ns-radio value="end">end</ns-radio>
    <ns-radio value="baseline">baseline</ns-radio>
    <ns-radio value="stretch">stretch</ns-radio>
  </ns-radio-group>
  <ns-space :align="align" style="margin-top: 12px">
    <ns-button size="s">按钮1</ns-button>
    <ns-button size="m">按钮2</ns-button>
    <ns-button>按钮3</ns-button>
  </ns-space>
</template>

<script lang="ts" setup>
import { ref } from 'vue'

const align = ref('start') as any
</script>

排列方向 ​

您可以通过 dir 属性来设置元素排列方向。可选值：vertical、horizontal、vertical-reverse、horizontal-reverse。默认为 horizontal。

<template>
  <p>内置尺寸：</p>
  <ns-radio-group v-model="dir" type="button">
    <ns-radio value="horizontal">horizontal</ns-radio>
    <ns-radio value="vertical">vertical</ns-radio>
    <ns-radio value="horizontal-reverse">horizontal-reverse</ns-radio>
    <ns-radio value="vertical-reverse">vertical-reverse</ns-radio>
  </ns-radio-group>
  <ns-space :dir="dir" class="margin" inline>
    <ns-button>按钮1</ns-button>
    <ns-button>按钮2</ns-button>
    <ns-button>按钮3</ns-button>
  </ns-space>
</template>

<script lang="ts" setup>
import { ref } from 'vue'

const dir = ref('horizontal') as any
</script>

换行 ​

您可以通过 wrap 属性来设置是否换行。

<template>
  <p>反转换行：</p>
  <ns-space wrap="reverse" :size="[10, 20]" class="margin" style="width: 300px">
    <ns-button v-for="item in 10" :key="item">按钮</ns-button>
  </ns-space>
</template>

溢出 ​

您可以通过 overflow 属性来设置溢出情况。

<template>
  <ns-space overflow="auto" class="margin" style="width: 400px">
    <ns-button v-for="item in 10" :key="item" style="margin: 12px 0">
      按钮
    </ns-button>
  </ns-space>
</template>

API ​

注意：对于默认值，例如：start，并不需要设置，只需要设置其他值即可。

参数描述类型默认值
inline是否内联。booleanfalse
dir排列方向。可选值：vertical、horizontal、vertical-reverse、horizontal-reversestringhorizontal
size间距尺寸。可选值：ss、s、m、l、xl。注意：对于 number 类型则默认为 px 单位。如果想自定义单位需要使用 string 类型。number | string | (number | string)[]l
justify水平对齐。可选值：start、center、end、space-between、space-aroundstringstart
align垂直对齐。可选值：start、center、end、baselinestringstart
overflow溢出处理。可选值：visible、hidden、clip、scroll、autostring | string[]true
wrap换行。boolean | wrap-reversetrue
