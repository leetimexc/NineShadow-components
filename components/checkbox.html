Checkbox 多选框 ​

在一组备选项中进行多选。

基础用法 ​

单独使用可以表示两种状态之间的切换。

<template>
  <div class="demo-checkbox">
    <ns-checkbox v-model="checked1">选项1</ns-checkbox>
    <ns-checkbox v-model="checked2">选项2</ns-checkbox>
  </div>
</template>

<script setup>
import { ref } from "vue";

const checked1 = ref(true);
const checked2 = ref(false);
</script>

<style scoped>
.demo-checkbox .ns-checkbox {
  margin-right: 20px;
}
</style>

禁用状态 ​

多选框不可用状态。设置 disabled 属性即可。

<template>
  <div class="demo-checkbox">
    <ns-checkbox v-model="checked1" disabled>禁用且选中</ns-checkbox>
    <ns-checkbox v-model="checked2" disabled>禁用且未选中</ns-checkbox>
  </div>
</template>

<script setup>
import { ref } from "vue";

const checked1 = ref(true);
const checked2 = ref(false);
</script>

<style scoped>
.demo-checkbox .ns-checkbox {
  margin-right: 20px;
}
</style>

多选框组 ​

适用于多个勾选框绑定到同一个数组的情景，通过是否勾选来表示这一组选项中选中的项。

<template>
  <div class="demo-checkbox">
    <ns-checkbox-group v-model="checkList">
      <ns-checkbox label="选项A"></ns-checkbox>
      <ns-checkbox label="选项B"></ns-checkbox>
      <ns-checkbox label="选项C"></ns-checkbox>
      <ns-checkbox label="禁用" disabled></ns-checkbox>
      <ns-checkbox label="选中且禁用" disabled></ns-checkbox>
    </ns-checkbox-group>
  </div>
</template>

<script setup>
import { ref } from "vue";

const checkList = ref(["选项A", "选中且禁用"]);
</script>

中间状态 ​

indeterminate 属性用以表示 checkbox 的不确定状态，一般用于实现全选的效果。

<template>
  <div class="demo-checkbox">
    <ns-checkbox
      v-model="checkAll"
      :indeterminate="isIndeterminate"
      @change="handleCheckAllChange"
    >
      全选
    </ns-checkbox>
    <div style="margin: 15px 0"></div>
    <ns-checkbox-group
      v-model="checkedItems"
      @change="handleCheckedItemsChange"
    >
      <ns-checkbox v-for="item in items" :key="item" :label="item">
        {{ item }}
      </ns-checkbox>
    </ns-checkbox-group>
  </div>
</template>

<script setup>
import { ref, watch } from "vue";

const checkAll = ref(false);
const isIndeterminate = ref(true);
const items = ["选项A", "选项B", "选项C", "选项D"];
const checkedItems = ref(["选项A", "选项B"]);

const handleCheckAllChange = (val) => {
  checkedItems.value = val ? items : [];
  isIndeterminate.value = false;
};

const handleCheckedItemsChange = (value) => {
  const checkedCount = value.length;
  checkAll.value = checkedCount === items.length;
  isIndeterminate.value = checkedCount > 0 && checkedCount < items.length;
};
</script>

可选项目数量的限制 ​

使用 min 和 max 属性能够限制可以被勾选的项目的数量。

<template>
  <div class="demo-checkbox">
    <ns-checkbox-group v-model="checkedItems" :min="1" :max="2">
      <ns-checkbox v-for="item in items" :key="item" :label="item">
        {{ item }}
      </ns-checkbox>
    </ns-checkbox-group>
  </div>
</template>

<script setup>
import { ref } from "vue";

const items = ["选项A", "选项B", "选项C", "选项D"];
const checkedItems = ref(["选项A", "选项B"]);
</script>

      <ns-checkbox
        v-model="checked3"
        label="选项1"
        border
        disabled

      <ns-checkbox
        v-model="checked4"

        border
        disabled
      ></ns-checkbox>
    </div>
  </div>

<script setup>
import { ref } from "vue";

const checked1 = ref(true);
const checked2 = ref(false);
const checked3 = ref(true);
const checked4 = ref(false);
</script>

<style scoped>
.demo-checkbox .ns-checkbox {
  margin-right: 20px;

按钮样式 ​

按钮样式的多选组合。只需要把 ns-checkbox 元素替换为 ns-checkbox-button 元素即可。

<template>
  <div class="demo-checkbox">
    <div>
      <ns-checkbox-group v-model="checkboxGroup1">
        <ns-checkbox-button v-for="item in options" :key="item" :label="item">
          {{ item }}
        </ns-checkbox-button>
      </ns-checkbox-group>
    </div>
    <div style="margin-top: 20px">
      <ns-checkbox-group v-model="checkboxGroup2" disabled>
        <ns-checkbox-button v-for="item in options" :key="item" :label="item">
          {{ item }}
        </ns-checkbox-button>
      </ns-checkbox-group>
    </div>
  </div>
</template>

<script setup>
import { ref } from "vue";

const options = ["选项A", "选项B", "选项C"];
const checkboxGroup1 = ref(["选项A"]);
const checkboxGroup2 = ref(["选项A"]);
</script>

不同尺寸 ​

Checkbox 组件提供除了默认值以外的三种尺寸：large、default、small。

<template>
  <div class="demo-checkbox">
    <div>
      <ns-checkbox-group v-model="checkboxGroup1" size="large">
        <ns-checkbox-button v-for="item in options" :key="item" :label="item">
          {{ item }}
        </ns-checkbox-button>
      </ns-checkbox-group>
    </div>
    <div style="margin-top: 20px">
      <ns-checkbox-group v-model="checkboxGroup2">
        <ns-checkbox-button v-for="item in options" :key="item" :label="item">
          {{ item }}
        </ns-checkbox-button>
      </ns-checkbox-group>
    </div>
    <div style="margin-top: 20px">
      <ns-checkbox-group v-model="checkboxGroup3" size="small">
        <ns-checkbox-button v-for="item in options" :key="item" :label="item">
          {{ item }}
        </ns-checkbox-button>
      </ns-checkbox-group>
    </div>
  </div>
</template>

<script setup>
import { ref } from "vue";

const options = ["选项A", "选项B", "选项C"];
const checkboxGroup1 = ref(["选项A"]);
const checkboxGroup2 = ref(["选项A"]);
const checkboxGroup3 = ref(["选项A"]);
</script>
