Input 输入框 ​

通过鼠标或键盘输入字符。

基础用法 ​

使用v-model实现双向绑定。

<script lang="ts" setup>
import { ref } from 'vue'

const input = ref('')
</script>

<template>
  <div>
    <ns-input v-model="input" placeholder="请输入内容" />
    <div style="margin-top: 15px">当前输入值: {{ input }}</div>
  </div>
</template>

禁用状态 ​

通过 disabled 属性指定是否禁用 input 组件。

<script lang="ts" setup>
import { ref } from 'vue'

const input = ref('禁用状态的输入框')
</script>

<template>
  <div>
    <ns-input v-model="input" disabled placeholder="请输入内容" />
  </div>
</template>

可清空 ​

使用 clearable 属性即可得到一个可清空的输入框。

<script lang="ts" setup>
import { ref } from 'vue'

const input = ref('可清空内容')
</script>

<template>
  <div>
    <ns-input v-model="input" clearable placeholder="请输入内容" />
    <div style="margin-top: 15px">当前输入值: {{ input }}</div>
  </div>
</template>

密码框 ​

使用 show-password 属性即可得到一个可切换显示隐藏的密码框。

<script lang="ts" setup>
import { ref } from 'vue'

const input = ref('')
</script>

<template>
  <div>
    <ns-input v-model="input" show-password placeholder="请输入密码" />
  </div>
</template>

带图标的输入框 ​

带有图标标记输入类型。

<script lang="ts" setup>
import { ref } from 'vue'

const input1 = ref('')
const input2 = ref('')
</script>

<template>
  <div class="demo-input-suffix">
    <span class="demo-input-label">使用 prefix 插槽</span>
    <ns-input v-model="input1" placeholder="请选择日期">
      <template #prefix>
        <ns-icon icon="calendar" />
      </template>
    </ns-input>
  </div>
  <div class="demo-input-suffix" style="margin-top: 15px">
    <span class="demo-input-label">使用 suffix 插槽</span>
    <ns-input v-model="input2" placeholder="请输入内容">
      <template #suffix>
        <ns-icon icon="search" />
      </template>
    </ns-input>
  </div>
</template>

<style scoped>
.demo-input-label {
  display: inline-block;
  width: 120px;
  text-align: right;
  margin-right: 10px;
}
.demo-input-suffix {
  margin-bottom: 15px;
}
</style>

文本域 ​

用于输入多行文本信息，通过将 type 属性的值指定为 textarea。

<script lang="ts" setup>
import { ref } from 'vue'

const textarea = ref('')
</script>

<template>
  <div>
    <ns-input
      v-model="textarea"
      type="textarea"
      placeholder="请输入内容"
      :rows="2"
    />
  </div>
</template>

复合型输入框 ​

可前置或后置元素，一般为标签或按钮。

<script lang="ts" setup>
import { ref } from 'vue'

const input1 = ref('')
const input2 = ref('')
const input3 = ref('')
</script>

<template>
  <div>
    <ns-input v-model="input1" placeholder="请输入内容">
      <template #prepend>Http://</template>
    </ns-input>
  </div>
  <div style="margin-top: 15px">
    <ns-input v-model="input2" placeholder="请输入内容">
      <template #append>.com</template>
    </ns-input>
  </div>
  <div style="margin-top: 15px">
    <ns-input v-model="input3" placeholder="请输入内容">
      <template #prepend>Http://</template>
      <template #append>.com</template>
    </ns-input>
  </div>
</template>

尺寸 ​

使用 size 属性改变输入框大小。除了默认大小外，还有另外两个选项：large, small。

<script lang="ts" setup>
import { ref } from 'vue'

const input1 = ref('')
const input2 = ref('')
const input3 = ref('')
</script>

<template>
  <div class="demo-input-size">
    <ns-input v-model="input1" size="large" placeholder="请输入内容" />
    <div style="margin: 15px 0">
      <ns-input v-model="input2" placeholder="请输入内容" />
    </div>
    <ns-input v-model="input3" size="small" placeholder="请输入内容" />
  </div>
</template>

<style scoped>
.demo-input-size .ns-input {
  width: 100%;
}
</style>

Input API ​

Props ​

名称说明类型可选值默认值
type类型stringtext / textareatext
model-value / v-model绑定值string / number——
size输入框尺寸stringlarge / default / smalldefault
placeholder输入框占位文本string——
clearable是否可清空boolean—false
show-password是否显示切换密码图标boolean—false
disabled是否禁用boolean—false
readonly原生 readonly 属性boolean—false
autocomplete原生 autocomplete 属性stringon / offoff
autofocus原生 autofocus 属性boolean—false
form原生 form 属性string——

插槽 ​

插槽名说明
prefix输入框头部内容，只对 type="text" 有效
suffix输入框尾部内容，只对 type="text" 有效
prepend输入框前置内容，只对 type="text" 有效
append输入框后置内容，只对 type="text" 有效

事件 ​

事件名说明回调参数
input在 Input 值改变时触发(value: string | number)
change仅在输入框失去焦点或用户按下回车时触发(value: string | number)
focus在 Input 获得焦点时触发(event: FocusEvent)
blur在 Input 失去焦点时触发(event: FocusEvent)
clear在点击由 clearable 属性生成的清空按钮时触发—

方法 ​

方法名说明参数
focus使 input 获取焦点—
blur使 input 失去焦点—
select选中 input 中的文字—
clear清空 input 的内容—
