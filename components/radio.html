NsRadio 单选框 ​

基本用法 ​

这里展示基本用法

<template>
  <ns-space dir="vertical">
    <ns-radio v-model="radio" value="1">单选框</ns-radio>
    <ns-radio v-model="radio" value="1" :disabled="true">禁用</ns-radio>
  </ns-space>
</template>
<script lang="ts" setup>
import { ref } from 'vue'
const radio = ref('')
</script>

控制状态 ​

通过 v-model (modelValue) 属性控制是否选中

<template>
  <ns-space dir="vertical">
    <ns-radio v-model="radioValue" :value="active">选框</ns-radio>
    <ns-button @click="handleChange">切换状态</ns-button>
  </ns-space>
</template>
<script setup lang="ts">
import { ref } from 'vue'
const active = ref(true)
const radioValue = ref(false)
const handleChange = () => {
  radioValue.value = !radioValue.value
}
</script>

单选框组 ​

基本单选框组使用方式，使用原则，当选项大于 5 个时 推荐使用选项框。使用ns-radio-group组件将ns-radio组件包裹使用

<template>
  <ns-radio-group v-model="radio" direction="horizontal">
    <ns-radio value="love">Love</ns-radio>
    <ns-radio value="and">And</ns-radio>
    <ns-radio value="peace">Peace</ns-radio>
  </ns-radio-group>
</template>
<script lang="ts" setup>
import { ref } from 'vue'
const radio = ref('')
</script>

    {{ radio }}
    <ns-radio value="love">Love</ns-radio>
    <ns-radio value="and">And</ns-radio>
    <ns-radio value="peace" disabled>Peace</ns-radio>

</template>

const radio = ref('love')
</script>

Radio API ​

Radio Attributes ​

参数描述类型默认值
value(v-model)选中项绑定值number ｜ string ｜ boolean--
label单选框的值number ｜ string ｜ boolean--
disabled按钮是否被禁用booleanfalse

Radio Slots ​

参数描述
default自定义内容

Radio Events ​

事件名说明类型
change绑定值变化时触发的事件Function

RadioGroup Attributes ​

参数描述类型默认值
value(v-model)选中项绑定值number ｜ string ｜ boolean--
solid当 type='button'时，按钮是否填充booleanfalse
type单选框组的类型'radio'｜'button''radio'
size单选框组的尺寸、间距's'｜ 'm'｜ 'l'｜'xl''l'
disabled单选框组是否被禁用booleanfalse
direction单选框的方向'horizontal'｜'vertical''horizontal'
